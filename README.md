## 1 difference between var let and const
In a javascript var let and const used for declare variable but they have some key differences

1. var is global scoped, and it's declared again and again
2. let is block scoped and it's introduce in ES6 , and it's value changeable
3. const also block scoped and introduce in ES6, but it's value not changeable

## 3 difference between map() filter() and and forEach()
1. map() return a new array
2. filter() also return new array but condition wise
3. forEach() is normal method like for of loop and it's doesn't return a new array;
